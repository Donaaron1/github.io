# Project 1

**Title:** [Sales Revenue and Passenger Ticketing Dashboard](https://github.com/Donaaron1/github.io/blob/main/Sales%20Revenue%20and%20Passenger%20Ticketing%20Dashboard.pbix)

**Tools Used:** Power BI (Power Query, DAX, Data Modelling, Filters, slicers, Visualization)

**Project Description:** This project involved analysing transaction data of a regional rail operator to improve customer satisfaction and operational efficiency. It is designed to provide a comprehensive overview of key performance metrics. The dashboard allows stakeholders to easily spot operational bottlenecks, refine timetables, and implement targeted strategies for better on-time performance and passenger satisfaction. The dashboard includes the following features:
Payment Method Preferences: 
Ticket Type: 
Sales by Purchase Channel: 
Refund Request VS Journey Status: 
Railcard Usage: 
Additionally, the dashboard includes interactive slicers and a timeline for:
Month: Filter the data to view performance for a specific month or range of months.
Year: Filter the data to view performance by year.
Refund Request and Journey Status Correlation: Focus on how the refund request is determined by journey status (delayed, cancelled, on time).


**Key findings:** 
Most Popular Ticket Types: Analyzed what ticket types are generating the most revenue and profit, aiding in optimization of price and availability to boost revenue and ticket sales.
Purchase Channel Profitability: Identified the most profitable sales channel, aiding in marketing decisions.
Refund Request and Journey Status Correlation: Highlighted the correlation between refund demands and journey status, helping to assess potential revenue loss and improve customer experience.
Preferred Payment Method: Evaluated customer payment behaviour for potential process improvements.
Price Variation Trends: Revealed trends or differences in ticket costs based on sales methods, allowing for more strategic planning.
Railcard Usage Insights: Determined the value of railcard promotions or potential marketing strategies.
